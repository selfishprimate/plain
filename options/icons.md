# Icons

Icon libraries and systems that provide consistent visual language for actions, navigation, and content types. Each library offers different styles, coverage, and implementation methods.

Choose an icon system that matches your design style and technical requirements:

### Lucide
The community-driven fork of Feather Icons with over 1000+ clean, consistent icons.
- **Style:** Outline, minimalist, geometric
- **Count:** 1000+ icons
- **Best for:** Modern interfaces, SaaS products, minimalist designs
- **Features:** Tree-shakeable, accessible, customizable stroke width

### Heroicons
Beautiful hand-crafted SVG icons by the makers of Tailwind CSS.
- **Style:** Outline and solid variants, rounded corners
- **Count:** 290+ icons (each in outline and solid)
- **Best for:** Tailwind projects, modern web apps, clean interfaces
- **Features:** React/Vue components, perfectly balanced, Tailwind optimized

### Phosphor Icons
Flexible icon family with multiple weights and styles for every icon.
- **Style:** Thin, light, regular, bold, fill, duotone
- **Count:** 850+ icons (x6 variants each)
- **Best for:** Design systems, apps needing weight variations
- **Features:** Variable weights, duotone option, extensive coverage

### Tabler Icons
Free and open source icons designed with perfect pixel alignment.
- **Style:** Outline, consistent 24px grid
- **Count:** 3000+ icons
- **Best for:** Dashboards, admin panels, data-heavy interfaces
- **Features:** Pixel perfect, extensive set, consistent stroke

### Feather Icons
The original simple and beautiful open source icons.
- **Style:** Outline, minimalist
- **Count:** 280+ icons
- **Best for:** Minimal designs, simple interfaces
- **Features:** Lightweight, clean aesthetic, widely recognized

### Font Awesome
The web's most popular icon library with extensive coverage.
- **Style:** Solid, regular, light, duotone, brands
- **Count:** 2000+ free, 16000+ pro icons
- **Best for:** Projects needing maximum coverage, brand icons
- **Features:** Brand icons, extensive categories, font or SVG

### Material Icons
Google's official icon set following Material Design principles.
- **Style:** Filled, outlined, rounded, two-tone, sharp
- **Count:** 2000+ icons
- **Best for:** Material Design apps, Google-style interfaces
- **Features:** Multiple themes, variable font option, official Google support

### Bootstrap Icons
Official open source icon library for Bootstrap.
- **Style:** Outline with some filled variants
- **Count:** 1800+ icons
- **Best for:** Bootstrap projects, traditional web apps
- **Features:** Bootstrap integration, SVG sprite support

### Remix Icon
Open source neutral-style icon library with both line and fill versions.
- **Style:** Line and fill variants
- **Count:** 2500+ icons
- **Best for:** Versatile projects, Chinese market apps
- **Features:** Business icons, comprehensive set, CDN ready

### Iconoir
Free open source library with personality and unique style.
- **Style:** Outline with rounded corners, playful
- **Count:** 1300+ icons
- **Best for:** Creative projects, modern startups
- **Features:** Unique personality, no attribution required

### Radix Icons
Crisp icons designed specifically for UI development.
- **Style:** 15x15px outline icons
- **Count:** 300+ icons
- **Best for:** Radix UI projects, small interface elements
- **Features:** Small size optimized, consistent weight, UI-focused

### Simple Icons
Brand icons for popular companies and services.
- **Style:** Monochrome brand logos
- **Count:** 2400+ brand icons
- **Best for:** Social links, tech stack displays, integrations
- **Features:** Official brand colors included, constantly updated

### Ionicons
Premium open source icons for Ionic Framework and web apps.
- **Style:** iOS, Material Design, outline styles
- **Count:** 1300+ icons
- **Best for:** Mobile apps, Ionic projects, cross-platform
- **Features:** Platform-specific variants, mobile optimized

### Fluent Icons
Microsoft's modern icon system for Windows and Office.
- **Style:** Regular and filled, rounded corners
- **Count:** 1700+ icons
- **Best for:** Windows apps, Microsoft-style interfaces
- **Features:** Fluent Design alignment, accessibility focused

### CSS.gg
Pure CSS icons without any dependencies.
- **Style:** Minimalist, geometric, pure CSS
- **Count:** 700+ icons
- **Best for:** Lightweight projects, CSS-only builds
- **Features:** No JavaScript needed, customizable via CSS

---

*Icons are the universal language of interfaces. Choose a library that provides comprehensive coverage, matches your visual style, and integrates smoothly with your tech stack.*
