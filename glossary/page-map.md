# Page Map

Common page structures and navigation patterns for different types of applications. Each pattern represents a proven approach to organizing content and user flows.

## Page Structures

Choose a page structure that matches your application type and user needs:

### SaaS Application
Standard structure for software-as-a-service products.
- **Landing Page** - Hero, features, pricing, testimonials, CTA
- **Sign Up/Sign In** - Authentication with OAuth options
- **Dashboard** - Overview metrics, recent activity, quick actions
- **Settings** - Profile, account, billing, integrations, notifications
- **Billing** - Plans, payment methods, invoices, usage
- **Features** - Core product pages (varies by product)
- **Help/Support** - Documentation, contact, chat widget

### E-commerce Platform
Essential pages for online selling and shopping.
- **Home** - Featured products, categories, promotions
- **Product Listing** - Filters, sorting, pagination, grid/list view
- **Product Detail** - Gallery, description, reviews, add to cart
- **Shopping Cart** - Item management, pricing, checkout CTA
- **Checkout** - Shipping, payment, order review
- **Account** - Orders, addresses, wishlists, preferences
- **Search Results** - Filtered results with faceted navigation

### Corporate Website
Professional presence for businesses and organizations.
- **Home** - Value proposition, services overview, social proof
- **About** - Company story, mission, team, values
- **Services/Products** - Detailed offerings with benefits
- **Case Studies** - Client success stories, results, testimonials
- **Blog/Resources** - Thought leadership, guides, whitepapers
- **Contact** - Form, location, hours, multiple contact methods
- **Careers** - Open positions, culture, benefits

### Content Platform
Publishing and content consumption focused structure.
- **Home Feed** - Personalized or algorithmic content stream
- **Explore/Discover** - Categories, trending, recommended
- **Article/Post** - Content display, comments, sharing
- **Author Profile** - Bio, published content, followers
- **Search** - Full-text search with filters
- **Bookmarks/Library** - Saved content organization
- **Settings** - Preferences, notifications, privacy

### Portfolio/Personal
Showcase work and personal brand.
- **Home** - Introduction, featured work, call-to-action
- **Work/Projects** - Case studies with process and outcomes
- **About** - Background, skills, experience, photo
- **Services** - What you offer, how you work, pricing
- **Blog** - Thoughts, tutorials, industry insights
- **Contact** - Form, email, social links, availability

### Educational Platform
Learning management and course delivery.
- **Home** - Featured courses, categories, benefits
- **Course Catalog** - Browse, filter, search courses
- **Course Detail** - Syllabus, instructor, reviews, enrollment
- **Learning Dashboard** - Progress, upcoming lessons, achievements
- **Lesson View** - Video/content, resources, discussions
- **Assessments** - Quizzes, assignments, certifications
- **Profile** - Achievements, certificates, learning history

### Community/Social
User-generated content and interaction.
- **Feed** - Timeline, stories, posts from network
- **Profile** - User info, posts, followers, following
- **Explore** - Discover users, content, trending topics
- **Messages** - Direct messaging, group chats
- **Notifications** - Activity alerts, mentions, interactions
- **Create** - Post composer, media upload, scheduling
- **Settings** - Privacy, security, preferences

### Documentation Site
Technical documentation and knowledge base.
- **Home** - Getting started, quick links, search
- **Guides** - Step-by-step tutorials, how-tos
- **API Reference** - Endpoints, parameters, examples
- **Components** - UI component documentation
- **Examples** - Code samples, playgrounds, demos
- **Changelog** - Version history, updates, migrations
- **Support** - FAQ, community, contact

### Marketplace
Two-sided platform for buyers and sellers.
- **Home** - Featured listings, categories, how it works
- **Browse** - Category navigation, filters, map view
- **Listing Detail** - Photos, description, seller info, actions
- **Seller Profile** - Ratings, listings, verification
- **Create Listing** - Multi-step form, media upload, pricing
- **Messages** - Buyer-seller communication
- **Transactions** - Orders, payments, disputes

### Admin Dashboard
Internal tools and management interfaces.
- **Overview** - KPIs, charts, recent activity
- **Analytics** - Detailed metrics, reports, exports
- **Users** - User management, roles, permissions
- **Content** - CMS, moderation, publishing
- **Settings** - System configuration, integrations
- **Logs** - Activity logs, audit trail, debugging
- **Tools** - Utilities, bulk actions, imports/exports

## Navigation Patterns

Common navigation approaches for different contexts:

### Top Navigation
Horizontal menu bar with dropdowns.
- **Best for:** Corporate sites, simple structures
- **Items:** 5-7 primary items maximum
- **Mobile:** Hamburger menu or priority+ pattern

### Sidebar Navigation
Vertical menu on left or right side.
- **Best for:** Applications, dashboards, documentation
- **Behavior:** Fixed, collapsible, or responsive
- **Mobile:** Off-canvas drawer or bottom nav

### Tab Navigation
Horizontal tabs for section switching.
- **Best for:** Settings pages, profile sections, filtered views
- **Behavior:** Underline, background, or pill style
- **Mobile:** Scrollable or dropdown

### Bottom Navigation
Mobile-first navigation at screen bottom.
- **Best for:** Mobile apps, PWAs
- **Items:** 3-5 primary actions
- **Enhancement:** FAB (Floating Action Button) for primary action

### Breadcrumb Navigation
Hierarchical path showing current location.
- **Best for:** Deep hierarchies, e-commerce, documentation
- **Display:** Home > Category > Subcategory > Page
- **Mobile:** Truncated or single parent link

### Mega Menu
Large dropdown panels with multiple columns.
- **Best for:** E-commerce, large catalogs, complex structures
- **Content:** Links, images, featured items
- **Mobile:** Full-screen overlay or accordion
